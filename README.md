# Generate QR code

Genera QR code para proyecto LMZ

### Prerequisites

python3.8

### Installing

Para configurar en entrono ejecutar:

En Linux
```
make.sh
```

## Comman Line

### Exportar imagen con QR al proyecto LMZ

Linux
```
usage: generateQR.sh
```

Este script usa para exportar QR code del proyecto LMZ
